Pseudo Code: Swap Two Numbers

1. Start
2. Declare variables: a, b, temp
3. Input values for a and b
4. Print "Before swapping: a =", a, "b =", b
5. Perform swap:
   a. Set temp = a
   b. Set a = b
   c. Set b = temp
6. Print "After swapping: a =", a, "b =", b
7. End

Example:
If a = 5 and b = 10, the output will be:

Before swapping: a = 5, b = 10
After swapping: a = 10, b = 5
